# OpenBetterSwarm - Planner Progress Bubble

OpenBetterSwarm features a **Planner Progress Bubble** that provides real-time visual feedback for the agent planning stage, surfaced natively in the chat UI via WebSockets.

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/MAI-AMAN/OPENBETTERSWARM.git
   cd OPENBETTERSWARM
   ```
2. Run the application:
   ```bash
   bash run.sh
   ```

This will automatically install dependencies and start the backend, frontend, and the electron shell.
